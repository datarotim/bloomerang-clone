## Dashboard Widgets (Confirmed from Press Release)

The Bloomerang Home page dashboard contains these confirmed widgets:

### 1. Recent Constituents
- Quick-add functionality
- Shows recently added/modified constituents

### 2. Revenue Metrics
- Tracked by **week, month, and fiscal year**
- Total donations display

### 3. Donor Retention Rate
- Visual retention display (referred to as "Donor Retention Wheel" in help docs)
- **Customization options** available
- Shows overall rate, first-time vs repeat

### 4. First-Time Donor Calls
- Recommended contacts to call
- Automatically shows **top 5** first-time donors
- Includes gift details directly on dashboard
- Can delegate outreach to team members

### 5. Reports
- Shows most recent reports
- Quick access to report builder

### 6. Campaign Tracking
- Customizable date ranges
- Progress indicators (bars)
- Goal vs. raised amounts

### 7. Task Management
- Displays assignments for users and teams
- Task completion tracking

## Constituent Profile Structure (Confirmed from Academy PDF)

### Header (persistent across all tabs)
- Constituent name
- Type: Individual or Organization
- Profile picture (pencil icon to edit)
- Quick-action dropdown
- Assigned groups

### Tab 1: Summary
- Giving Summary
- Recent Timeline
- Engagement Level (Cold → Cool → Warm → Hot → On Fire)
- Generosity Score (same scale)
- Relationships
- Website visit history (below engagement meter, if enabled)

### Tab 2: Profile
- Information area (contact details, demographics)
- Edit / Merge / Change Type / Delete actions
- Custom Fields
- Created / Last Modified timestamps

### Tab 3: Timeline
- Chronological touchpoint view
- Timeline view OR list view toggle
- Filter by type (right side)
- Year navigation (right side)
- Highlights for major moments
- Filter: "Interactions (No Mass Emails)"
- Icon types: donations, emails, calls, events, notes, letters, tasks, soft credits

### Tab 4: Relationships
- Household relationships
- Employer relationships
- Other database relationships

## Engagement Level Scale
| Level | Visual | Description |
|-------|--------|-------------|
| Cold | Blue indicator | Minimal interaction |
| Cool | Light blue | Some engagement |
| Warm | Yellow/amber | Regular engagement |
| Hot | Orange | High engagement |
| On Fire | Red/flame | Extremely engaged |

Algorithm by Dr. Adrian Sargeant. Factors: giving, volunteerism, marketing engagement, email opens/clicks, event attendance.

---

## Generosity Score Scale
Same visual scale as Engagement Level (Cold → On Fire).
Based on DonorSearch publicly available wealth screening data.
Appears on constituent/household Summary screen.
