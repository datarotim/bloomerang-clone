## 1. SIDEBAR NAVIGATION (Left Side)

**Source: SoftwareConnect Constituent Summary & Dashboard screenshots (actual product screenshots)**

The left sidebar is a narrow vertical panel on the left side of the screen with the following structure:

### Top of Sidebar
- **Bloomerang logo** (green leaf/spark icon + "bloomerang" text in dark gray, lowercase) at the very top
- A **search bar** spans the top of the main content area (not in the sidebar): "Search for constituents" with a magnifying glass icon

### Sidebar Menu Items (top to bottom, each with an icon on the left)
1. **Dashboard** - house/home icon (highlighted green when active, with green left border indicator)
2. **Constituents** - person/people icon
3. **Reports** - bar chart icon
4. **Letters** - envelope/letter icon (with a right-arrow/play triangle indicator)
5. **Emails** - email icon
6. **Social Hub** - connected circles/social icon
7. **Data Tools** - database/gear icon
8. **Settings** - gear/cog icon
9. **Kindful** - Kindful logo icon (integration link at the bottom)

### Sidebar Styling
- **Background**: White/very light gray
- **Text color**: Dark gray (#535862 or similar)
- **Active item**: Green text (#3F9107 or similar), green left border highlight, slightly lighter background
- **Icons**: Simple line-style icons, gray when inactive, green when active
- **Collapse**: The sidebar has a collapse icon attached to it (not the logo); collapsing hides text labels but keeps icons visible
- **Width**: Approximately 160-180px expanded

### Top-Right Navigation Bar Icons
- **Help/question mark** icon (green circle with "?")
- **Notifications/bell** icon
- **User profile/avatar** icon (green circle with person silhouette)

## 3. CONSTITUENT PROFILE PAGE

**Source: SoftwareConnect Constituent Summary + TrustRadius screenshots (actual product screenshots)**

### Header Bar (Constituent Info)
- **Profile photo/avatar** placeholder (gray circle with person silhouette icon)
- **Constituent name** in large bold dark text (e.g., "Jay Barclay Love" or "Sarah Jane Smith")
- Below name: Household name (e.g., "Love, Jay and Christie"), constituent number (e.g., "#1" or "#3")
- Email address as clickable link in teal/green
- **Address** section: Street, City, State, ZIP (green clickable text)
- **Phone** section: "Work (866) 332-2999"
- **Relationship Manager** section: Manager name (only visible in some views)
- **Dropdown chevron** button (teal/green circle with white down arrow) at far right of header

### Tab Navigation (below header)
Four tabs in a horizontal row:
1. **Summary** - green/teal filled button when active (white text on green background, rounded)
2. **Profile** - gray text when inactive
3. **Timeline** - green/teal filled button when active
4. **Relationships** - gray text when inactive

Active tab styling: Green background (#4AA74B or similar teal-green), white text, rounded rectangle
Inactive tab styling: Gray text, no background

### Summary Tab Layout (Two columns)

#### Left Column (wider, ~60-65%):

**Giving Summary Widget**
- Title: "Giving Summary" in large dark text
- Toggle buttons at top-right: **Revenue** (green filled), **Raised** (blue/teal filled), **Soft Credits** (blue/teal filled)
  - Revenue button: green background with white text
  - Raised/Soft Credits: teal/blue background with white text
- **Area/line chart** showing giving over time:
  - X-axis: Years (e.g., '13, '14, '15 ... '22)
  - Y-axis: Dollar amounts (e.g., 56000, 112000)
  - Solid green line with filled green area underneath (Revenue)
  - Dashed blue line (Raised or Soft Credits)
  - Circle data points at each year
- Below chart - key metrics in a grid:
  - **Lifetime**: Two green badges showing amounts (e.g., "$576,502.50" green, "$610,252.50" lighter green)
  - **Average**: Two green badges
  - **First Transaction**: Amount and date in green text
  - **Latest Transaction**: Amount and date in green text
  - **Largest Transaction**: Amount and date in green text
- Window controls: minimize (-), maximize (square), close (X) icons in green at top-right of widget

**Recent Timeline Activity**
- Title: "Recent Timeline Activity"
- List of recent events, each showing:
  - Green gift icon (for payments) or blue speech bubble icon (for interactions)
  - Date and amount
  - Transaction type (e.g., "Recurring Payment", "Split Payment")

#### Right Column (narrower, ~35-40%):

**Engagement Level Widget**
- Red heart icon + "Engagement Level" title
- Horizontal bar/meter:
  - Gradient bar from cool (left) to hot (right)
  - When fully engaged: solid **red** bar
  - Scale: Cold -> Warm -> Hot -> On Fire!
- Status text below: "On Fire!" (or "Cold", "Warm", "Hot")

**Online Engagement Widget**
- Globe/web icon + "Online Engagement" title
- "Website Visits (Last 30 Days)" label
- Visit count number
- "Most Recent Visit" label
- "X Days Ago" in green text

**Generosity Widget**
- Dataro icon (teal) + "Generosity" title
- "Update" button (gray outlined, rounded)
- Horizontal bar meter:
  - Segmented red/gradient bar showing generosity level
  - Scale similar to engagement (levels of giving potential)
- Status text: "On Fire!" (or other levels)
- Large teal button: **"View Generosity Details"** (teal background, white text, rounded)

**Relationships Widget**
- "Relationships" section header
- List of related constituents

- **Primary sans-serif**: "PP Right Grotesk" (Bloomerang's marketing site)

- **Product UI**: Appears to use a clean sans-serif (likely system fonts or similar to Helvetica/Arial/Open Sans based on screenshot rendering)
- **Font weights**: 300 (light), 400 (regular), 500 (medium), 700 (bold), 900 (black)

### Typography Patterns
- **Dashboard greeting**: Large, regular weight, dark text
- **Widget headers**: Small, uppercase, letter-spaced, gray (e.g., "AMOUNT RAISED", "DONOR RETENTION")
- **Dollar amounts**: Large bold (appears serif or heavy weight) dark text
- **Metric details**: Small, regular weight, gray text
- **Constituent names**: Medium weight, green/teal color when clickable
- **Tab labels**: Regular weight; active = white on green background
- **Sidebar nav items**: Regular weight, dark gray; active = green text

## 13. KEY OBSERVATIONS FOR UI RECONSTRUCTION

1. **The sidebar is always visible** on desktop, with icon + text label for each nav item
2. **Dashboard uses a card/widget grid layout** - not a strict grid, but responsive cards arranged in rows
3. **White cards with subtle shadows** are the primary container pattern throughout
4. **Green is the dominant accent color** - used for active states, links, chart fills, buttons
5. **Uppercase gray headers** are used consistently for widget/section titles with letter-spacing
6. **The constituent profile uses a tabbed interface** with Summary, Profile, Timeline, Relationships
7. **Timeline is a central visual metaphor** - vertical timeline with icons and cards
8. **Engagement and Generosity use horizontal bar meters** - not circular gauges
9. **The Donor Retention widget uses a donut/ring chart** - green on white
10. **Typography is clean and modern** - sans-serif throughout the product UI
11. **Cards have consistent padding** and spacing between widgets
12. **The search bar** is prominent at the top of every page, centered above the main content
